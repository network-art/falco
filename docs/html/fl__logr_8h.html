void fl_logr_cfg_priority ( int  priority)

Configure the priority for the logger.

Configure/Set the priority for the logger. The default priority is the log level LOG_INFO used in syslog.

Parameters
[in]priorityPriority or syslog log level. Acceptable values are: 
LOG_EMERG (highest)
LOG_ALERT
LOG_CRIT
LOG_ERR
LOG_WARNING
LOG_NOTICE
LOG_INFO (Default)

void fl_logr_openlog ( const char *  ident)

Open a connection to the system logger.

Open a connection to the system logger for the application. The string pointed to by ident is prepended to every message, and is typically set to the program name.

The following options are passed to the system logger: LOG_CONS | LOG_NDELAY | LOG_PID, LOG_LOCAL0.

Parameters
[in]identTypically the program or the application name
See also
Documentation for openlog() describes the options passed to the system logger.
