Allocate dynamic memory.

This function uses calloc() to allocate memory for an arrary of _n_ elements of size @ sizeof(_casttotype_) and returns a pointer to the allocated memory.

Parameters
[in]_casttotype_Used to determine the size of each element, and the data-type to which the allocated memory needs to be casted to.
[in]_n_Number of elements
[in]_var_Variable to which the allocated memory is assigned to
[in]_msg_Message to be logged in case there is a failure in allocating memory
See also
calloc(3)
